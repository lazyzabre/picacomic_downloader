# 哔咔漫画收藏夹下载程序  
<font color=red>由于哔咔更新了API加密，此项目已不再有效</font>
哔咔APP的本子下载是乱序的，这个程序用来直接下载收藏夹里的文件。  
> 版本1.3更新，修复了因为特殊字符导致的部分漫画无法下载问题,和杀毒软件报错问题  
> 版本1.2更新，修复了token过期导致的无法登录，和40章后无法下载的问题  
> 版本1.1更新，增加了多选下载和暂停
